# 소비 규율 · 승격/강등 절차

> 계층이 있어도 소비 규율이 없으면 컴포넌트는 장식이 된다. "있는 걸 쓰게 만드는" 메커니즘의 정본.

## 전형적 실패 모드 (이 규율이 없을 때 반드시 일어나는 일)

Input primitive가 아이콘 슬롯까지 지원하는 프로젝트에서도, 소비 규율이 없으면 검색 인풋이 십수 곳에서 제각각 재조립된다 — raw input / 아이콘 왼쪽 / 아이콘 오른쪽 / 전용 래퍼가 공존하고, 동일한 pill 스타일 문자열이 여러 파일에 복붙되며 주석으로 "다른 화면과 통일"을 **수동 유지**하게 된다. 이스케이프 해치(REPLACE형 class prop)가 있으면 반복 패턴은 variant로 승격되는 대신 호출부마다 해치로 재구현된다. 결과: 어느 쪽이 의도된 디자인인지 아무도 모르는 드리프트 고고학.

## 소비 규율 5조

1. **그리기 전 매니페스트** — `arch:manifest --layer primitive`(항상) + `--domain <d>`(해당 시)를 실행해 읽는다. "비슷한 게 있는지 몰랐다"는 사유가 아니라 절차 위반.
2. **있으면 소비, 모자라면 variant** — 우선순위: ① primitive에 variant/prop 추가 ② 독립 역할이면 새 primitive ③ 진짜 일회성 모양일 때만 이스케이프 해치(최후 수단). 호출부 덮어쓰기부터 시작하지 않는다.
3. **Rule of Two** — 같은 시각 패턴의 **두 번째** 구현 시점 = 승격 시점. UI는 셋째를 기다리는 동안 서로 다르게 드리프트한다.
4. **이스케이프 해치는 일회성** — REPLACE형 class prop(`inputClass`류)의 계약: 설정 시 그 영역 스타일을 호출자가 전부 소유. 같은 해치값(클래스 토큰 정렬 비교)이 2파일 = 위반(`DUPLICATE_ESCAPE_HATCH`). 해치의 합법 상주지는 **primitive 내부뿐**(새 primitive가 기존 primitive를 조립하며 내부에서 쓰는 것 — 그 값이 한 곳에만 존재하게 되므로).
5. **primitive에 도메인 기본값 금지** — 판정: 그 기본 문구가 다른 제품에서 말이 안 되면 도메인 어휘. 처방: 기본값 삭제 → required prop → 타입 에러가 나는 모든 호출부가 문구 공급 책임자.

## 승격 절차 (composite 조각 → primitive) — 6단계 한 커밋

1. **역할명** — 도메인을 지운 이름(`FooSearchBar` ✗ → `SearchInput` ✓). 이름에서 도메인이 안 지워지면 승격 대상이 아니라 도메인 부품.
2. **요구사항 매트릭스** — 기존 손구현 전 콜사이트 나열, (시각 변형 × 인터랙션 변형) 표. variant는 이 표에서만 도출 — 미래 추측 variant 금지.
3. **작성** — kit 템플릿 기반: 토큰만, 도메인 기본값 없이, a11y·IME 내장, @component+TSDoc 앵커, 스토리/스펙.
4. **전 콜사이트 이관** — 같은 커밋에. "만들어두고 점진 이관"은 스타일 계열을 하나 더 늘리는 것.
5. **재발방지 룰** — 그 패턴의 손구현을 잡는 감사 룰을 `.svelte-arch/config.mjs`에 같은 커밋으로 추가. 프로젝트 룰 목록 = 승격의 역사.
6. **잔여 래퍼 정리** — 기존 전용 래퍼는 (a) 도메인 문구만 남긴 얇은 composite로 축소 또는 (b) 소비처 1곳이면 인라인 후 삭제.

슬롯형이냐 세트형이냐는 constitution.md §3.10 판별표(신호 2개 이상일 때만 세트).

## 강등 (primitive → composite)

primitive에서 도메인 지식 발견 시: prop으로 끌어올릴 수 있으면 prop화 후 잔류, 구조 자체가 특정 화면 전용이면 해당 도메인 폴더로 이동(접미사 `.composite`로 리네임 — git 이력에 사건으로 남음).

## 고아 관리

매니페스트 소비처 0 = ⚠고아 표시. 즉시 삭제 금지(승격 직후·선행 구축 중일 수 있음) — 두 릴리스 연속 고아면 삭제 검토. 대체된 고아(신형 컴포넌트가 대체)는 즉시 삭제.
